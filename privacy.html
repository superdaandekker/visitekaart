Privacyverklaring

We beschermen jouw gegevens en houden alles transparant.

Welke data slaan we op?

Deze MVP draait volledig in je browser. We bewaren je kaartgegevens in localStorage zodat je wijzigingen behouden blijven. Er wordt geen cloud of externe database gebruikt.

De enige data die in deze demo wordt opgeslagen, zijn de velden die je zelf invult op de pagina “Mijn kaart”.

Hoe kun je data verwijderen?

  • Via de app

    Klik op Uitloggen om alle opgeslagen kaartgegevens te verwijderen.

  • Via de browser

    Wis je browserdata (localStorage) voor deze site om helemaal opnieuw te beginnen.
